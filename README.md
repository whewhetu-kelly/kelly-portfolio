# Premium Portfolio

A world-class personal portfolio website built with React 19, Vite, Tailwind CSS, and Framer Motion.

## Features

- Premium glassmorphism design
- Dark/Light mode toggle
- Smooth Framer Motion animations
- Interactive project showcase
- Responsive across all devices
- SEO optimized
- Static site ready for Vercel deployment

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- React Router
- Framer Motion
- React Icons
- Swiper.js

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Deploy to Vercel

```bash
vercel --prod
```
